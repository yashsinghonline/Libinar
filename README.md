# 📚 Libinar

> **Discover. Read. Learn.**
>
> **Libinar** is a modern, open-source digital library that lets anyone explore and read thousands of free public domain books through a fast, beautiful, and distraction-free reading experience. Built using the **Project Gutenberg** and **Open Library** APIs, Libinar combines classic literature with a contemporary interface, making timeless knowledge accessible to everyone.

---

## 🌟 About

Libinar is designed to solve a common problem: most free online libraries are either outdated, difficult to navigate, or cluttered with unnecessary interfaces. Instead of overwhelming users, Libinar focuses on delivering a clean, intuitive, and enjoyable reading experience inspired by modern web applications.

Readers can instantly search for books, browse by category or author, explore detailed book pages, and start reading within seconds—all without creating an account or downloading additional software.

Whether you're a student, researcher, casual reader, or someone rediscovering the classics, Libinar provides a simple gateway to thousands of legally available books from trusted public sources.

# 🌍 Data Sources

Libinar does **not** host books itself.

Instead, it aggregates publicly available information from trusted open knowledge projects.

## Project Gutenberg

Provides:

- Public domain books
- HTML versions
- EPUB files
- Plain text books
- Downloadable formats
- Author information

Project Gutenberg is one of the world's largest collections of public domain literature.

---

## Open Library

Provides:

- Book covers
- Additional metadata
- ISBN information
- Subject classifications
- Author details
- Edition information

Open Library is an initiative of the Internet Archive that aims to create a web page for every published book.

# 💡 Why Libinar?

Unlike traditional online libraries, Libinar focuses on user experience first.

Instead of presenting hundreds of filters and outdated layouts, it offers an interface that feels modern, fast, and intuitive.

The goal is simple:
